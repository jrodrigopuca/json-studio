JSON Spark is a Chrome extension designed with privacy as a priority. **All data is processed locally in your browser and never transmitted to external servers.**

## Core Principles

### üîí Local Processing

- **All JSON data is processed exclusively on your device**
- The extension does not send any data to remote servers
- There is no backend infrastructure or server that collects information

### üö´ No Personal Data Collection

- **We do not collect, store, or process personal data**
- We do not track your browsing activity

- We do not maintain usage history
- We do not use tracking cookies
- We do not employ third-party analytics tools

### üíæ Local Storage

- Data you save in the "Saved" view is stored only in your browser's local storage (localStorage)
- This data remains on your device and never syncs with external services
- You have full control to delete this data at any time
- Local storage is isolated by extension and not accessible by other websites

### üåê No External Connections

JSON Spark operates completely autonomously:

- Does not make calls to external APIs
- Does not download resources from remote servers
- Does not communicate with third-party services
- All functionality is included in the extension package

## Extension Permissions

The extension requests the following Chrome permissions solely to function correctly:

- **`storage`**: To save user preferences (theme, settings) and saved JSONs locally in your browser
- **`activeTab`**: To detect and format JSON files in the current tab
- **Access to JSON URLs**: To intercept and visualize JSON responses when you navigate to .json URLs

**None of these permissions are used to collect, transmit, or share data.**

## Data That Remains on Your Device

The following data is stored exclusively in your browser:

1. **User preferences**: theme (dark/light), font size, preferred indentation
2. **Saved JSONs**: JSON files that you choose to save in the "Saved" view
3. **Extension configuration**: extension preferences and custom options

## Your Rights

As a JSON Spark user, you have full control over your data:

- **Access**: You can view all stored data through Chrome Developer Tools
- **Deletion**: You can delete all data by uninstalling the extension or clearing local storage
- **Portability**: You can export your saved JSONs using the download function
- **Control**: No data leaves your device without your explicit action (such as downloading a file)

## Open Source

JSON Spark is open source software under the GNU General Public License v3.0:

- Source code is publicly available for audit
- You can verify that no data collection mechanisms exist
- The community can review and contribute to the code

## Extension Updates

Extension updates are distributed through the Chrome Web Store:

- Updates only modify the extension code
- Do not alter, access, or transmit your locally stored data
- Do not introduce telemetry or tracking mechanisms

## Security

We implement measures to protect privacy:

- All processing is local, eliminating data interception risks
- No third-party scripts are used that could compromise privacy
- The extension operates in the browser's sandboxed mode
- No remote code is executed nor external resources loaded

## Contact

If you have questions or concerns about this privacy policy, you can:

- Review the project source code
- Open an issue on the GitHub repository
- Contact the developer through the project repository

## Changes to This Policy

Any changes to this privacy policy will be:

- Documented in the repository version control
- Communicated in version release notes
- Updated with a new "Last updated" date

**This policy can only become more restrictive, never less. JSON Spark will maintain its commitment to not collect or transmit user data.**
